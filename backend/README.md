# AssetFlow AI Backend

A complete Node.js + Express + MongoDB backend for an AI-powered enterprise asset and resource management system.

## Features
- JWT authentication with register/login/profile
- Asset, employee, assignment, maintenance, dashboard, report, notification, and AI endpoints
- Auto-generated codes for employees, assets, assignments, and maintenance
- QR code generation for assets
- Centralized error handling and validation

## Scripts
- npm install
- npm run dev
- npm start

## Environment
Set the following variables in .env:
- PORT
- MONGO_URI
- JWT_SECRET
